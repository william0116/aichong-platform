# 🐾 爱宠平台

公益驱动的宠物生态服务平台 - 让每一只流浪猫都拥有温暖的家

## 项目简介

爱宠平台首创"C端宠主APP + B端门店管理 + 流浪动物公益救助"三位一体模式，以公益驱动商业，以商业反哺公益，构建可持续发展的宠物生态闭环。

## 核心业务

- **公益救助服务** - TNR绝育、紧急救助、伤病治疗、临时寄养、公益领养
- **宠物便民服务** - 电子档案、智能健康提醒、线上预约、用品商城
- **政企联动服务** - 街道社区对接、动保协会协作、城市治理项目
- **公益生态服务** - 公益商城、公益社区、志愿调度平台

## 核心优势

- 🔄 **双端闭环优势** - C端APP + B端门店 + 公益救助三位一体
- 📊 **全周期数字档案** - 宠物从家养到救助的全生命周期记录
- ❤️ **公益驱动运营** - 公益爱心值体系创新会员运营模式
- 🚀 **轻量化可拓展** - 基础版低价入门 + AI问诊/IoT等增值模块

## 技术架构

- 三层微服务架构：用户APP、商家后台、运营端三端协同
- 全链路数据安全：AES加密、RBAC权限、MFA认证
- 场景定制化模块：TNR、流浪救助等行业专属功能

## 本地运行

直接在浏览器中打开 `index.html` 文件即可预览：

```bash

open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```
